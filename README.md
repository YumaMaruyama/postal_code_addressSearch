# postal_code_addressSearch

## [アプリ概要]
郵便番号を入力すると郵便番号でわかる住所の情報が表示されます。
郵便番号検索APIを使用しています。
* アプリURL： http://133.167.92.108:8081/zipcode

## [アプリ説明] ##

郵便番号だけで住所検索ができる便利なアプリです。
適当に知識なしに7桁打ち込んで住所が表示されればかなりの強運の持ち主です。
実際に検索することが可能です。  
* アプリURL：http://133.167.92.108:8081/zipcode
